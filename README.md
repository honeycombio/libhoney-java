# libhoney-java [![Build Status](https://circleci.com/gh/honeycombio/beeline-java.svg?style=shield)](https://circleci.com/gh/honeycombio/beeline-java) [![Maven Central](https://img.shields.io/maven-central/v/io.honeycomb.libhoney/libhoney-java.svg)](http://search.maven.org/#search%7Cga%7C1%7Cg%3Aio.honeycomb.libhoney%20a%3Alibhoney-java)

Java library for sending events to [Honeycomb](https://honeycomb.io), a service for debugging your software in production.

Requires Java 7 or greater.

- [Usage and Examples](https://honeycomb.io/docs/connect/java)
- [API Reference](https://honeycombio.github.io/libhoney-java/)

## Local Build

Build with Java 7 and above by running the following from the root directory:
```
./mvnw package
```
